# 🕰️ geol

`geol` is a CLI tool to display **End-of-Life (EOL) dates** for your stack,
powered by the [endoflife.date](https://endoflife.date) API.

### 🚀 Quick Start

```shell
# List all available products
docker run --rm optnc/geol list

# Check EOL status for a specific product
docker run --rm optnc/geol product java

# Export all EOL data to a DuckDB database
docker run --rm -e GEOL_API_DELAY_MS=200 -v $(pwd):/data -w /data optnc/geol export
```

### 🖥️ Supported Platforms

- `linux/amd64`
- `linux/arm64`
