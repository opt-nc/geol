# Fonctionnalités de `Markdown`

La documentation de `geol` prend en charge les fonctionnalités Markdown standard (GitHub Flavored Markdown).

## La chaîne d'outils `pandoc`

`pandoc` est un convertisseur de documents permettant de transformer du Markdown en EPUB, PDF, DOCX, HTML, et plus encore. Flux de travail typique :

Installez `pandoc`. Voir https://pandoc.org/installing.html pour plus de détails.

- Produire `geol-report.md` avec `geol`.

Exemples — d'autres formats de sortie sont possibles :
```bash
pandoc geol-report.md -o geol-report.epub
```
```bash
pandoc geol-report.md -o geol-report.pdf --pdf-engine=xelatex
```
